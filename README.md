# コザ高女子テニス部 九州インターハイ遠征資金造成プロジェクト

令和8年度 沖縄県インターハイ予選にて準優勝を果たし、長崎県で開催される九州インターハイへの出場権を獲得したコザ高等学校女子テニス部の遠征および活動資金を造成するための独立プロジェクトです。

## 📋 プロジェクト概要

*   **対象団体**: コザ高等学校 女子テニス部
*   **目標**: 九州インターハイ遠征費の確保および日々の部活動運営費（ボール代等）の補填
*   **背景**: PTA会費の不足に伴う部活動援助金の削減により、選手個人の負担が大幅増。持続可能な活動基盤を整えるため、地域・OG・サポーターからの支援を募る。
*   **遠征期間**: 2026年（令和8年）6月18日 〜 6月23日（5泊6日）
*   **開催地**: 長崎県（九州インターハイ）
*   **受付締切**: 2026年6月30日（火）
